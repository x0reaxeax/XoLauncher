# ![XoLauncher](https://i.imgur.com/reKe6vG.png) XoLauncher
### 1. Setting up XoLauncher  
   **1.1** Download [XoLauncher](https://github.com/x0reaxeax/XoLauncher/releases/download/rls/XoLauncher.exe)  
   **1.2** Open **XoLauncher** and navigate the first-time setup to your MultiMC directory  
### 2. Launching Minecraft  
   **2.1** Select your MultiMC instance and press `Launch Minecraft`  


![XoLauncher](https://i.imgur.com/sMWxf93.jpg)

## Info
  * XoLauncher automatically syncs XoPack mods with the server, so you don't have to move a finger if we ever add a new mod!
  * That's about it actually..  
  * XoLauncher requires a functional MultiMC instance with **`Forge 14.23.5.2816`** (compatible up-to `14.23.5.2838`)
  * XoPack is a **REALLY BIG** modpack (over 100+ mods)! You might need a beefy computer to run it..
